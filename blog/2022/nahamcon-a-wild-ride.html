    Writeup For The "A Wild Ride" Challenge In NahamCon CTF 2022
    writeup
    2 May 2022 ● 10 min read

    Introduction

    “A Wild Ride” is a challenge in NahamCon CTF that took place from the 28th of April to the 30th of April.

    Challenge

    I've got this encrypted ZIP file filled with .gpx'es, and I just know there's a message in there...

    And we are provided a download link for gpx.zip.

    From the challenge description, we can see that there are a bunch of gpx files.

    Cracking The Zip File

    Trying to unzip gpx.zip with unzip gpx.zip prompts us for a password, which we don’t know. We can crack this password with John The Ripper and rockyou.txt. Install John The Ripper on your platform (either compile it or download the binaries). You also need zip2john.c. Compile it with gcc zip2john.c -o zip2john. Download rockyou.txt, we will use this as our wordlist.

    We can first convert it to a format John The Ripper can understand with zip2john. ./zip2john gpx.zip > forjohn.txt will give the zip file in the correct format for John The Ripper in forjohn.txt.

    We can now start cracking the zip file using rockyou.txt as the wordlist with

    john forjohn.txt --wordlist=rockyou.txt

    After a few seconds, John The Ripper identifies the password as crackme. We can now unzip the file using crackme as the password with unzip gpx.zip and supplying crackme as the password.

    We can organize them by creating a new folder named ride or anything you prefer and moving all .gpx files to that folder with

    mkdir ride; mv *.gpx ride

    Merging The Files

    We can use gpxmerge from gpx-cmd-tools to merge all the .gpx files. We can install gpx-cmd-tools with

    python3 -m pip install gpx-cmd-tools

    Now we can merge all the gpx files with

    gpxmerge *.gpx -o ../merged.gpx

    This creates merged.gpx in the previous directory.

    Plotting The File

    You can open any GPX plotter you like, here we will use gpx.studio. Click on Load GPX and open merged.gpx. This will render the merged GPX file revealing the flag, flag{gpx_is_cool}.
