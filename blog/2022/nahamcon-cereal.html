    Writeup For The Cereal Challenge In NahamCon CTF 2022
    writeup
    1 May 2022 ● 7 min read

    Introduction

    “Cereal” is a challenge in NahamCon CTF that took place from the 28th of April to the 30th of April.

    Challenge

    Oh no I dropped my cereal!!

    And we are provided a download link for mystery.sal. Oh no I dropped my cereal!! is not really helpful.

    Exploring The File

    Personally, I have no idea what this file is. We can see what file in Linux tells us. file mystery.sal tells us it’s a Zip archive. mystery.sal: Zip archive data, at least v2.0 to extract, compression method=deflate.

    We can unzip this file like any other .zip file resulting in these files

    [compromyse@spoofy cereal] $ unzip mystery.sal
Archive:  mystery.sal
  inflating: digital-0.bin           
  inflating: digital-1.bin           
  inflating: digital-2.bin           
  inflating: meta.json               
[compromyse@spoofy cereal] $ ls
digital-0.bin  digital-1.bin  digital-2.bin  meta.json  mystery.sal
[compromyse@spoofy cereal] $

    Running file on one of these .bin files tells us it is data. Not very useful. We can open this in your favorite hex editor, I like hexdump. So running hexdump -C digital-0.bin gives <SALEAE> as the header.

    Searching for SALEAE in your favorite search engine results in Saleae Logic Analyzer coming up in the results. We can download their software to check out these files from here for your Operating System, mine Linux.

    Solving The Challenge

    We can launch the Saleae Logic Analyzer software. On the home page we see an Open a capture option. This is probably what we want.

    We can open mystery.sal in it. Zooming in to one of the white blocks, results in the flag being printed above it.

    The bit.ly link redirects to this video. Funny.
